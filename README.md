## How To Use
1. First step Clonning Git
```
git clone https://github.com/justnaf/kurirmu-laravel.git
```
2. Composer And NPM Install
```
composer install
npm install
```
3. rename .env.example to .env
4. run this
```
php artisan migrate
php artisan key:generate
```
